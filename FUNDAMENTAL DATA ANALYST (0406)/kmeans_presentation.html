Latar Belakang
  • Penyakit kardiovaskular adalah penyebab utama kematian global (17,9 juta/tahun - WHO)
  • Kolesterol tinggi merupakan faktor risiko utama penyakit kardiovaskular
  • Identifikasi dini kelompok berisiko tinggi sangat penting untuk pencegahan
  • K-Means Clustering dapat membantu mengidentifikasi pola dan kelompok pasien
  • Dataset Cardiovascular Disease dari Kaggle: 70.000 pasien dengan data komprehensif
Rumusan Masalah & Tujuan

Tujuan Penelitian

  • Eksplorasi data kolesterol pasien
  • Menentukan jumlah cluster optimal
  • Mengelompokkan pasien dengan K-Means
  • Memberikan rekomendasi medis

Temuan Penting:

Terdapat korelasi positif yang kuat antara level kolesterol dengan prevalensi penyakit kardiovaskular:

  • Kolesterol normal: ~40% memiliki penyakit kardiovaskular
  • Kolesterol tinggi: ~60% memiliki penyakit kardiovaskular
  • Kolesterol sangat tinggi: ~70% memiliki penyakit kardiovaskular
Penentuan Jumlah Cluster Optimal

Elbow Method

Mencari "siku" pada grafik inertia vs K. Elbow terlihat di sekitar K=3.

Silhouette Score

K=3 memiliki score 0.289 (kategori "cukup" dengan overlap moderat).

Keputusan: K = 3

  • Trade-off baik antara simplicity dan kualitas
  • Interpretable untuk konteks medis (risiko rendah, sedang, tinggi)
  • Didukung oleh Elbow Method dan Silhouette Score

Evaluasi Model:

  • Silhouette Score: 0.289 (cluster terbentuk dengan overlap moderat)
  • Davies-Bouldin Index: 1.234 (separasi antar cluster memadai)
  • Calinski-Harabasz Score: Tinggi (dispersi antar-cluster baik)
Cluster 0: Risiko Rendah (34.4%)

Karakteristik

  • Usia rata-rata termuda: 51.2 tahun
  • Kolesterol paling rendah: 1.15 (mendekati normal)
  • Tekanan darah normal: 115/76 mmHg
  • Berat badan relatif lebih rendah: 68.5 kg
  • 89.2% memiliki kolesterol normal

Profil Kesehatan

  • Kondisi kesehatan paling baik di antara 3 cluster
  • Risiko kardiovaskular paling rendah
  • Gaya hidup kemungkinan lebih sehat

Rekomendasi

  • Maintenance gaya hidup sehat
  • Check-up rutin tahunan
  • Fokus pada pencegahan primer
Cluster 1: Risiko Sedang (43.1%)

Karakteristik

  • Usia menengah: 54.8 tahun
  • Kolesterol sedikit di atas normal: 1.35
  • Tekanan darah pre-hipertensi: 128/85 mmHg
  • Berat badan moderat: 74.8 kg
  • 78.1% kolesterol normal, 21.9% di atas normal

Profil Kesehatan

  • Kondisi kesehatan sedang dengan beberapa faktor risiko
  • Berada di ambang batas normal-abnormal
  • Perlu perhatian untuk mencegah progresivitas

Rekomendasi

  • Monitoring berkala setiap 3-6 bulan
  • Modifikasi gaya hidup (diet dan olahraga)
  • Edukasi tentang faktor risiko
  • Program kontrol berat badan
Cluster 2: Risiko Tinggi (22.5%)

Karakteristik

  • Usia paling tua: 58.3 tahun
  • Kolesterol tertinggi: 1.95 (hampir kategori 2)
  • Tekanan darah hipertensi stage 1: 145/96 mmHg
  • Berat badan paling tinggi: 82.3 kg
  • Hanya 48.7% kolesterol normal, 51.3% di atas normal/tinggi

Profil Kesehatan

  • Kondisi kesehatan paling berisiko
  • Multiple risk factors untuk kardiovaskular
  • Memerlukan intervensi medis segera

Rekomendasi

  • Monitoring ketat bulanan
  • Intervensi medis (kemungkinan medikasi: statin, antihipertensi)
  • Program penurunan berat badan intensif
  • Diet rendah kolesterol
  • Rujukan ke spesialis jantung
Kesimpulan
  • K-Means berhasil mengidentifikasi 3 kelompok pasien dengan profil risiko berbeda
  • Distribusi: 34.4% risiko rendah, 43.1% risiko sedang, 22.5% risiko tinggi
  • Usia, kolesterol, dan tekanan darah adalah faktor pembeda utama antar cluster
  • Terdapat korelasi positif yang kuat antara kolesterol tinggi dengan penyakit kardiovaskular
  • Model clustering memiliki kualitas cukup baik (Silhouette Score: 0.289)
  • Sekitar 22.5% pasien berada dalam kategori risiko tinggi dan memerlukan intervensi segera
  • Framework ini dapat digunakan untuk stratifikasi risiko dan personalisasi intervensi medis
Saran

Praktik Medis

  • Implementasi risk stratification berbasis clustering
  • Program monitoring intensif untuk cluster risiko tinggi
  • Alokasi resources sesuai tingkat risiko
